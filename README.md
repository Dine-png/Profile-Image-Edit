# Profile-Image-Edit

A local browser tool for previewing profile images across several social UI contexts before uploading them.

## Features

- Drag, zoom, rotate, and color-adjust a profile image
- Preview the result in X, Discord, and Instagram-style contexts
- Switch UI language between Korean, Japanese, and English
- Export final images as `400x400 PNG` or `800x800 PNG`

## Run

Open `index.html` in a browser.
